  1. OpenAI (ChatGPT) API 密钥获取：
  2. 打开 platform.openai.com 并登录你的 OpenAI 账号。
  3. 进入左侧菜单的「API keys」页面（也可直接访问 platform.openai.com/api-keys）。
  4. 点击「Create new secret key」创建一个新密钥。
  5. 复制生成的密钥（以 sk- 开头），粘贴到上方的 API Key 输入框。
  6. 注意：使用 API 会按 OpenAI 的用量计费，需要账户里有余额或绑定支付方式。
  7. Bytedance (字节) API 密钥获取：
  8. 打开 api.siliconflow.cn 并登录（或从字节官方渠道获取 API 平台地址）。
  9. 进入「API Keys」管理页面。
  10. 创建新密钥，复制后粘贴到上方输入框。
  11. 确保账户余额充足。

⚠️ 重要：API Key 等同于你账户的密码，泄漏后可能产生费用。请勿分享给他人、不要提交到代码仓库或截图外发。默认不会保存密钥；只有勾选“记住 API”时才会加密保存在本机。

  1. OpenAI (ChatGPT) API Key:
  2. Open platform.openai.com and sign in to your OpenAI account.
  3. Go to the "API keys" page from the left menu (or visit platform.openai.com/api-keys directly).
  4. Click "Create new secret key" to create a new key.
  5. Copy the generated key (starts with sk-) and paste it into the API Key field above.
  6. Note: API usage is billed by OpenAI; your account needs credit or a payment method.
  7. Bytedance (字节) API Key:
  8. Open api.siliconflow.cn or use the official Bytedance API platform link.
  9. Go to the "API Keys" management page.
  10. Create a new key and copy it to the field above.
  11. Ensure your account has sufficient balance.

⚠️ Important: Your API Key is like your account password and may incur charges if leaked. It is not saved by default; it is encrypted locally only when “Remember API” is selected.
